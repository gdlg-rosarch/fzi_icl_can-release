This package contains the FZI icl_can library wrapped as catkin compliant package.
It was developed by the the FZI Research Center for Information Technology in Karlsruhe, Germany.
Also it provides a build script to build the pcan library from peak systems,
which will then be included in binary releases.

See the website http://www.peak-system.com/fileadmin/media/linux/index.htm
for details on the pcan library.
